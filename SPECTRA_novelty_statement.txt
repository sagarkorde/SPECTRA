SPECTRA Novelty Statement — IEEE TIFS Submission
=================================================

We present SPECTRA (Spectral-Probabilistic Ensemble Clustering for Transaction
Recognition and Authentication), a novel hybrid framework for behavioral analysis
and authentication of blockchain transactions. Unlike prior works that treat
transaction pattern recognition as a static supervised classification task
(e.g., EvolveGCN [Pareja et al., AAAI 2020], BERT4ETH [He et al., WWW 2023]),
SPECTRA introduces four synergistic innovations absent from the literature.
First, we construct an address-level spectral graph representation using the
normalized graph Laplacian eigenvectors, providing topology-aware embeddings
that capture structural transaction roles without requiring node labels. Second,
we integrate a Variational Autoencoder with HDBSCAN density clustering in a
unified latent space, enabling simultaneous representation learning and
anomaly scoring without pre-specifying cluster count—a key limitation of
KMeans- and GMM-based approaches. Third, SPECTRA introduces a Bayesian trust
scoring mechanism with Beta-conjugate posterior updates, providing calibrated
per-address authenticity estimates with associated uncertainty bounds—a
capability absent from discriminative GNN baselines (AA-GNN [Fan et al.,
IEEE TII 2024]) and graph autoencoders (Liu et al., IEEE TDSC 2024). Fourth,
Wasserstein-distance-based drift detection over temporal cluster distributions
enables real-time behavioral shift alerting without retraining, addressing the
concept-drift challenge inherent in live blockchain monitoring. Collectively,
SPECTRA achieves state-of-the-art performance across all six transaction
categories while providing interpretable, uncertainty-quantified authentication
decisions suitable for forensic and compliance applications.

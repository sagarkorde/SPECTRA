# Checkpoints

`spectra_ckpt.pkl` (the full trained SPECTRA checkpoint, ~862MB) is not tracked
in this repository because it exceeds GitHub's 100MB per-file limit.

A smaller test checkpoint (`../checkpoints_test/spectra_ckpt.pkl`, ~5.7MB) is
included for pipeline verification.

To obtain the full trained checkpoint, open an issue on this repository or
contact the author directly.
